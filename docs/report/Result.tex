\section{結果}

湯気のシミュレーションの結果を図\ref{result}に示す。
実装では底面に一様な温度、水蒸気量を定義した結果である。
レンダリングには\cite{smoke3d}にある独自のレンダリングモジュールを用いた。
提案した手法を用いることで湯気の発生と消滅に近い映像を出力するできることを確認した。
